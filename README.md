# RestStatistic

Краткое описание проекта.

## Let business create company, employees, menu, make orders and collect statistic

## Установка

### Установка зависимостей

- **yarn**

### Заходим в директрию сервера

- **cd server**

### Иницализируем БД

- **yarn migrate**

### Запускаем сервер

- **yarn dev**

### Запускаем интерфейс Prisma

- **yarn prisma**
